# Superstore Data Analysis

**Dashinee Parmanum**  
*Self-directed portfolio hybrid project in Statistics and Data Analysis - Fall 2025*

---

This project demonstrates data cleaning, analysis, and visualization using Excel, Power BI and SQL on a U.S. Superstore sales dataset.

---

## Files and Folders
- **`Superstore_Datasets`:**
  - `superstore_US.csv` = Original raw dataset
- **`Superstore_Docs`:**
  - `Superstore_Report.md` = Detailed project overview and insights
  - `Superstore_Process.md` = Step-by-step methodology and formula explanations
  - `Superstore_Exercises.md` = Exercises/Questions
- **`Superstore_Images`** = screenshots
- **`Superstore_Excel`** = `#1_Excel.xlsx`
- **`Superstore_PowerBI`** = `#1_PowerBI.pbix`
- **`Superstore_SQL`** = `#1_SQL.sql`

## Key Features
- Comprehensive sales and profit analysis across products, sub-categories, and categories.
- Geographic performance mapping for regions, states, and cities.
- Time-series analysis of sales and profit trends on a monthly, quarterly, and yearly basis.
- Segment-level insights showing which customer segments drive the most revenue and profit.
- Assessment of discount impact on sales and profitability.
- Identification of underperforming products and categories.
- Analysis of seasonal trends and shipping mode efficiency.
- Top customer identification using SQL queries.
- Profit margin calculations to highlight high-margin products and categories.
- Integration of Excel, Power BI, and SQL workflows for multi-tool data analysis.

## Skills Demonstrated
- Data Cleaning & Transformation = Handling missing values, standardizing date formats, and preparing data for analysis.
- Data Analysis = Aggregation, sorting, filtering, and pivot tables in Excel; DAX formulas and calculated columns in Power BI; SQL querying in PgAdmin.
- Data Visualization = Line charts, scatter plots, maps, and bar charts to highlight trends and key insights.
- Statistical Thinking = Identifying correlations, trends, and performance metrics across multiple dimensions.
- Problem-Solving = Addressing data inconsistencies, import errors, and visualization challenges.
- Documentation & Reporting = Clearly explaining methodology, formulas, and analytical steps.

## Usage
- Open the `#1_Excel.xlsx` to explore pivot tables and summaries.
- Open `#1_PowerBI.pbix` to interact with dashboards, charts, and map visualizations.
- Run `#1_SQL.sql` in PgAdmin or another PostgreSQL environment to identify top customers and perform additional aggregation queries.
- Reference `Superstore_Process.md` for step-by-step methodology and formulas used in the analysis.
- Reference `Superstore_Report.md` for a narrative overview of insights and findings.
